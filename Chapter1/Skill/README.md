## Java Skill


### static context
在Java中，static是一個關鍵字，用於聲明類成員（字段、方法、代碼塊）為靜態成員。它也可以用於靜態內部類和靜態導入。        
        
static具有以下特點和行為：        
        
靜態成員屬於類本身，而不是類的實例。它們在類被加載時初始化，並且不需要類的實例即可訪問。        
        
靜態成員可以直接通過類名訪問，無需創建類的實例。    
        
靜態成員在內存中只有一份拷貝，被所有類的實例共享。      
        
靜態成員可以被類的所有實例訪問，也可以在類的靜態方法中訪問。
